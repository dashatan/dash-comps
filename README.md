This is my personal portfolio website built with Next.js, TypeScript, and Tailwind CSS, showcasing my projects, skills, and experience as a developer.

## 💻 Technologies

Here are some of the technologies I specialize in:

- **Frontend**: React, Next.js, TypeScript, JavaScript, Tailwind CSS
- **UI Libraries**: Shadcn UI, Radix UI, Embla Carousel
- **State Management**: React Query, React Hook Form
- **Backend**: Node.js, Prisma ORM
- **Languages**: JavaScript/TypeScript, HTML, CSS
- **Tools**: Git, ESLint, Prettier

## 🚀 Featured Projects

This portfolio includes demonstrations and information about various projects I've worked on, including:

- Web applications with responsive designs
- Component libraries with accessibility features
- Multilingual applications supporting English, Farsi, and Arabic
- Database-integrated applications using Prisma

## 🧩 Portfolio Structure

```
portfolio/
├── app/                  # Next.js app directory with main pages
├── components/           # Reusable UI components I've created
├── features/             # Feature sections (Home, Projects, About, etc.)
├── lib/                  # Utility functions and custom hooks
├── locales/              # Translation files (en, fa, ar)
├── public/               # Static assets and images
└── styles/               # Custom styles and Tailwind configuration
```
