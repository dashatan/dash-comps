# Internationalization (i18n)

This project supports multiple languages: English, Persian (Farsi), and Arabic.

## How to Use

### Language Toggle

The language toggle is available in the header component. It allows users to switch between English, Persian, and Arabic.

### Adding New Translations

To add new translations, update the `translations` object in `lib/language-context.tsx`:

```typescript
const translations = {
  en: {
    // English translations
    key: 'English text',
  },
  fa: {
    // Persian translations
    key: 'متن فارسی',
  },
  ar: {
    // Arabic translations
    key: 'النص العربي',
  },
};
```

### Using Translations in Components

To use translations in your components, import the `useLanguage` hook and use the `t` function:

```typescript
import { useLanguage } from '@/lib/language-context';

export function MyComponent() {
  const { t } = useLanguage();

  return (
    <div>
      <h1>{t('key')}</h1>
    </div>
  );
}
```

### RTL Support

The application automatically handles RTL (Right-to-Left) layout for Persian and Arabic languages. The following CSS classes are automatically adjusted for RTL:

- `ml-auto` / `mr-auto`
- `pl-8` / `pr-8`
- `ml-2` / `mr-2`

### Adding New RTL Styles

If you need to add more RTL-specific styles, you can add them to the `globals.css` file:

```css
html[dir='rtl'] .your-class {
  /* RTL-specific styles */
}
```

## Technical Details

- The language preference is stored in localStorage
- The HTML `lang` attribute is automatically updated based on the selected language
- The document direction (`dir` attribute) is automatically set to `rtl` for Persian and Arabic
- The application uses React Context for language state management
